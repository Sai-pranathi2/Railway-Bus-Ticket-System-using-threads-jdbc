## Railway/ Bus Ticket System using threads+jdbc
This project is a simple Railway/ Bus Ticket System developed using java,Threads and JDBC


##  Feautures
- Book Ticket
- Cancel TIcket
- View Ticket Details
- Multiple user booking using Threads
- Database storage using JDBC
## Languages used
-java

## Technologies used
- java
- JDBC
- Multithreading
- Oracle/MySQL

## Software Requirements
- Java JDK
- Eclipse/Intelli J
- Oracle/MYSQL
- Git

## Hardware Requirements
- Minimum 4GB RAM
- 500GB Hard Disk
- Windows/Linux OS
